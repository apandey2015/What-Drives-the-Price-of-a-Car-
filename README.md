# What-Drives-the-Price-of-a-Car-
The goal is to understand what factors make a used car more or less expensive.
# Overview
The project checks dataset with information on 3 million used cars. The provided dataset contains information on 426K cars to ensure the speed of processing. In the end, we need to provide clear recommendations to the client—a used car dealership—as to what consumers value in a used car.
